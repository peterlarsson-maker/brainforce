# ROADMAP v2

## Long-term memory system

- Persistent, searchable memory linked to user sessions
- Automatic summarization of conversations

## RAG / embeddings

- Support retrieval-augmented generation using external vector stores
- Plugin architecture for multiple embedding providers

## Multi-model support

- Ability to switch between and combine different LLMs
- Model selection based on cost, latency, or capability

## Role system expansion

- Granular permissions and custom roles for users
- Admin dashboard for role management

## Local AI integration

- Run lightweight models locally for offline operation
- Fallback to local model when network unavailable

## Enterprise features

- SSO and OAuth integrations (Azure AD, Okta, etc.)
- Audit logging and compliance reports
- Multi-tenant support with data isolation
