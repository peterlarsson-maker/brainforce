# Gem Local – Setup Guide

Detta paket hjälper dig att sätta upp en lokal AI-assistent kallad **Gem**.

---

## Krav

- Python 3.10+
- Git
- [Ollama](https://ollama.com/download)
- [AnythingLLM](https://github.com/Mintplex-Labs/anything-llm) (rekommenderad webbaserad chatt-UI)

---

## Installation

```bash
# 1. Installera Python och Git om du inte redan har det.
# 2. Installera Ollama från https://ollama.com/download

# 3. Klona AnythingLLM och följ dess installationsguide:
git clone https://github.com/Mintplex-Labs/anything-llm.git
cd anything-llm
# (Följ README för setup)
