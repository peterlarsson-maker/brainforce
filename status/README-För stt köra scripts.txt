För stt köra scripts


för att starta projectstatus.py, skriv i poowershell vid prompt:

python projectstatus.py

för att starta projectstatus-restore.py, skriv i poowershell vid prompt:

python projectstatus-restore.py


Så här kör du: tradstart.py

Se till att du har kört projektstatus.py minst en gång först!

Öppna terminal i status/tradstart/ Kör:

python tradstart.py
